# 04 데이터 시각화 진행 기록

## 2026-06-22 시각 품질 보정

- 기존 청록·보라·자홍색 브랜드 그라데이션에서 자홍색을 제거했다.
- 전역 팔레트를 남색·저채도 청록·중성 회색 중심으로 재정의했다.
- 상권 KPI와 차트의 자홍색 강조도 저채도 청록으로 통일했다.
- 데이터 도착 순서에 따라 환경 요약·상단 통계·대체 차트가 비어 보이던 초기화 경합을 수정했다.
- 데스크톱·모바일 공개 화면 10개 시각 기준선을 새 팔레트로 갱신하고 연속 재검증했다.

- 착수일: 2026-06-21
- 현재 상태: 로컬 구현·자동 검증 완료, 실데이터 대사 대기
- 선행 조건: 01·02 완료, 03의 검증된 데이터 계약만 사용

## 착수 원칙

03의 서울시 OpenAPI `8088` 실데이터 기준선은 아이폰 테더링 경로 제한 때문에 보류한다. 일반 유선 또는 Wi-Fi에서 재검증하기 전에는 외부 중계 서버를 선택하지 않는다. 04에서는 이미 실키·실DB 또는 공식 파일로 검증된 데이터와 샘플 계약만 사용하며, 생활시설 5종의 미확정 실데이터 수치를 시각화하지 않는다.

## 검증 결과

- 백엔드 JUnit 전체: 81건 통과
- 프런트엔드 구문 검사: 57개 파일 통과
- 프런트엔드 단위 테스트: 43건 통과
- 상호작용 E2E: 1440·1280·768·390px에서 33건 통과, 대상 외 3건 조건부 제외
- 공개 라우트 스모크 E2E: 24건 통과
- 중대 접근성 E2E: 20건 통과(홈 신규 차트 포함 4개 기준 폭 재검증)

## 다음 순서

1. 일반 유선/Wi-Fi 연결 후 03의 생활시설 실데이터 기준선을 확정한다.
2. 실데이터 기준으로 카드·표·다운로드 값을 최종 대사한다.

## 외부 판단 대기

- 일반 유선 또는 Wi-Fi 연결 후 서울시 OpenAPI `openapi.seoul.go.kr:8088` 재검증
