# 다국어 공개 정책

## 현재 결정

- 공개 언어는 한국어(`ko`)만 사용한다.
- 영어(`en`), 일본어(`ja`), 중국어 간체(`zh-CN`) 리소스는 검토 대기로 유지한다.
- 전체 화면 번역, 시설 고유명 폴백, 날짜·단위 표기, 접근성 문구 검토가 끝나기 전에는 공개 선택기에 표시하지 않는다.
- 실행 중 외부 자동번역 API를 호출하지 않는다.

## 공개 조건

1. 공통 셸과 6개 공개 화면 번역 완료
2. 시설 고유명 미번역 상태 표시 확인
3. 언어별 URL 공유·복원 및 PDF/CSV 표기 검증
4. 390px·768px·1440px 시각 회귀 통과
5. 내부 검토자가 해당 언어 공개를 승인

현재 `APPROVED_LANGUAGES=ko`가 기준이며, 미승인 언어 리소스는 코드에 존재하더라도 사용자 선택기에 노출하지 않는다.
