## 공간·배포 공통 기준

- 공식 통계와 기본 화면 범위는 `GEUMCHEON`이다.
- 경계 생활권은 `BORDER_AREA`, 외부 비교 원천은 `EXTERNAL_REFERENCE`로 분류하며 원천 행을 삭제하지 않는다.
- 이용조건이 확인되지 않은 데이터셋은 `SCREEN_ONLY`로 두고 다운로드와 외부 API 재배포를 모두 차단한다.
- 데이터셋별 접근 정책은 `DatasetAccessPolicyRegistry`에서 독립적으로 전환한다.
- 수집 실패나 품질 게이트 실패 시 기존 정상 스냅샷을 보존한다.

## 운영상 남은 외부 의존성

무더위쉼터 자동 수집에는 행정안전부 재난안전데이터 공유플랫폼에서 별도 활용 신청 후 발급되는 `SAFETY_DATA_API_KEY`가 필요하다. 키가 없을 때 수집기는 실패로 기존 데이터를 덮지 않고 정책상 건너뛴다.
