# 공개 장애 공지 템플릿

## 데이터 갱신 지연

> `[분야/데이터명]` 자료 갱신이 지연되고 있습니다. 현재 화면에는 `[마지막 정상 기준시각]`에 확인된 자료를 표시합니다. 시설 검색·목록 등 사용 가능한 기능은 계속 제공하며, 복구 확인 후 기준시각을 갱신하겠습니다.

## 지도 장애

> 지도 표시가 일시적으로 원활하지 않습니다. 시설 목록과 전화·상세 정보는 계속 이용할 수 있습니다. 위치 탐색이 복구될 때까지 목록을 이용해 주세요.

## 일부 기능 장애

> `[기능명]` 이용이 일시적으로 제한됩니다. 다른 공개 화면과 기존 정상 자료에는 영향이 없습니다. 발생시각은 `[시각]`, 다음 안내 예정시각은 `[시각]`입니다.

## 복구 완료

> `[기능/데이터명]`이 `[복구시각]`에 정상화됐습니다. 화면의 기준시각이 `[새 기준시각]`으로 갱신됐는지 확인했습니다. 지연 시간 동안 기존 정상 자료는 보존됐습니다.

공지에는 API 키, 내부 호스트, 상세 오류, 개인 연락처를 넣지 않는다. 영향 범위·마지막 정상 기준시각·다음 안내시각·대체 행동만 공개한다.
