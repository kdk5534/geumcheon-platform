# 핵심 과업 흐름

- 작성일: 2026-06-18
- 목적: 사용성 테스트와 화면 연결 기준 정리

## 1. 오늘의 금천에서 주의 정보 확인

1. 사용자는 `오늘의 금천`에 진입한다.
2. 상단 요약에서 상태 배지와 기준시각을 확인한다.
3. 주의 알림 또는 핵심 요약 카드에서 필요한 항목을 선택한다.
4. 필요 시 `분야별 > 안전·재난`의 최근 도시현황으로 이동한다.

성공 조건:

- 20초 이내에 현재 상태와 다음 행동을 파악할 수 있어야 한다.

## 2. 내 주변에서 시설 찾기

1. 사용자는 `내 주변`으로 이동한다.
2. 시설 카테고리를 선택한다.
3. 지도 또는 목록에서 가장 가까운 후보를 확인한다.
4. 연락 또는 상세 행동으로 이어진다.

성공 조건:

- 3번 이내 조작으로 원하는 시설 유형의 결과를 볼 수 있어야 한다.

## 3. 우리 동 비교 시작

1. 사용자는 `우리 동` 허브에 진입한다.
2. 비교할 동을 선택한다.
3. 선택 동의 접근성·인구·60세 이상 비중을 구 평균과 비교한다.
4. 필요하면 생활권 세부 비교 또는 인구 분석으로 이동한다.

성공 조건:

- 동 선택 후 같은 화면에서 구 평균과의 차이 한 가지를 설명할 수 있어야 한다.

## 4. 분야별 허브 탐색

1. 사용자는 `분야별` 허브에 진입한다.
2. 안전·재난, 교통·이동, 보건·복지, 교육·돌봄, 환경·생활, 지역경제·상권 질문 중 목적에 맞는 카드를 선택한다.
3. 대표 정보를 확인한 뒤 관련 서비스 또는 `원자료 찾기`로 이동한다.

성공 조건:

- 허브 카드만 보고도 각 화면의 용도를 구분할 수 있어야 한다.

## 5. 출처와 원본 데이터 확인

1. 사용자는 허브 또는 홈에서 `데이터 찾기`로 이동한다.
2. 검색 또는 필터로 필요한 데이터셋을 좁힌다.
3. 상세 미리보기에서 출처와 갱신 주기를 확인한다.

성공 조건:

- 2분 이내에 원하는 데이터셋의 출처와 갱신 정보를 찾을 수 있어야 한다.
