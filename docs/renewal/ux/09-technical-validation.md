# UI/UX·정보구조 기술 검증 기록

- 검증일: 2026-06-19
- 범위: `오늘의 금천`, `내 주변`, `우리 동`, 신규·기존 공개 URL
- 환경: 로컬 정적 서버, 브라우저 DOM·콘솔 검사, 샘플 데이터 모드

| 항목 | 결과 | 확인 내용 |

검증 중 브라우저 콘솔 오류는 확인되지 않았다.

## 2. 반응형

핵심 3화면을 390, 768, 1280, 1440px 너비에서 검사했다.

| 화면 | 390 | 768 | 1280 | 1440 |
| --- | --- | --- | --- | --- |
| 오늘의 금천 | 가로 넘침 없음 | 가로 넘침 없음 | 가로 넘침 없음 | 가로 넘침 없음 |
| 내 주변 | 가로 넘침 없음 | 가로 넘침 없음 | 가로 넘침 없음 | 가로 넘침 없음 |
| 우리 동 | 가로 넘침 없음 | 가로 넘침 없음 | 가로 넘침 없음 | 가로 넘침 없음 |

- 각 너비에서 빠른 행동, 지도·목록 영역, 동 선택 영역이 화면 안에 유지됐다.
- 1280px 화면의 200% 확대를 근사한 640 CSS px 재배치 검사에서도 세 화면 모두 문서 전체 가로 넘침이 없었다.
- 브라우저 실제 200% 확대, 스크린리더 낭독과 터치 기기 검사는 수동 검증으로 남긴다.

## 3. URL 호환

다음 직접 진입이 빈 화면 없이 목표 모듈과 1차 메뉴 상태를 유지했다.

- 신규: `#/home`, `#/nearby`, `#/dong`, `#/topics`, `#/datasets`
- 우리 동 세부: `#/dong?section=population&district=...`, `#/dong?section=accessibility&district=...`
- 기존 공개: `#/map`, `#/population`, `#/geo`, `#/realtime`, `#/indicators`, `#/commercial`, `#/catalog`, `#/about`
- 임시 운영 호환: `#/api`, `#/api-logs`, `#/admin`

임시 운영 호환 경로는 공개 메뉴에 노출하지 않는다. 별도 운영 엔트리와 인증 경계 구현은 `08-public-admin-boundary.md`와 02 단계에서 처리한다.

## 4. 남은 사람 중심 검증

기술 검사는 사용자 과업 성공을 대신하지 않는다. 다음 항목은 실제 참여자와 담당자 판단이 필요하다.

1. 주민·보호자 5~8명, 민원 담당자 2명, 데이터 관리자 1명 모집
2. 5초 테스트, 첫 클릭, 시설 찾기, 동 비교, 출처 찾기 수행
3. 과업 성공률 80%, 첫 클릭 성공률 70%, 주의 정보 확인 중앙값 20초 확인
4. 기준일·출처 발견률 90%와 샘플·지연 상태 이해 여부 확인
5. 브라우저 200% 확대, 키보드 전체 순서, 스크린리더 핵심 흐름 수동 검사

측정값이 없는 상태에서 목표를 통과했다고 기록하지 않는다.
